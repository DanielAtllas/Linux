Introdução - Linux

O Linux é um sistema operacional gratuito, o qual gera uma redução de custos e pirataria no ramo da computação (voltada a sistemas operacionais), além de possuir uma grande variedade de aplicações e várias distribuições disponpiveis e uma alta veracidade e segurança. Assim como é um sistema com mais estabilidade e com uma vasta performance, além de fornecer uma grande segurança à seus usuários. A redução do custo em softwares e a transparência nos sistemas operacions também são pontos a serem levados em consideração.

O Linux é a base de insfraestrutura em grandes empresas, como a Amazon.com e a Google. Assim como 70% dos servidores web usam o Linux, 30% dos sistemas embarcados também utilizam desta ferramenta.

  • Origem

    • Antes de se ter a versão definitiva do Linux, uma série de outras fases foram trabalhadas até a chegada do novo sistema operacional. Sendo essas fases:

    1. Unix
      2.

      Primeiro, teve a criação de um S.O. chamado de UNIX, sendo ele um dos primeiros sistemas operacionais criados no ano de 1965. O UNIX viria a ser um sistema operacional multitarefa e multiusuário, tendo sua primeira versão lançada em 1971. Desenvolvido e programado por Ken Thompson e Dennis Ritchie.


    2. GNU
      3.

      Após a ascenção do Unix, um nono projeto de S.O. foi orquestrado, sendo desenvolvido por Richard Stallman, o projeto GNU veio com o objetivo de ser mais completo que o UNIX e de livre acesso, mesmo sendo baseado no seu antecessor. Porém, com o defeito de que ainda faltava-se o KERNEL, o qual é a interface entre as aplicações do usuário e o hardware (gerenciamento de memória e processos, acesso a CPU e memória).


    3. Minix
      4.

      Já em 1987, o MINIX foi criado com o intuito de ser um sistema operacional voltado para estudos, também baseado no Unix. Sendo ele um S.O. reduzido, voltado para computadores pessoais, além de possuir um código fonte aberto, o qual foi desenvolvido e projetado por Andrew Tanenbaum.

    4. LINUX
      5.

      Por fim, em 1991 o projeto LINUX foi iniciado, sendo um sistema operacional livre para um computador específico, o AT 386.486 e este, foi baseado nas atribuições do Minix. Portado pelo BASH 1,08 - Interface de execução de comandos e GCC 1,40 - Compilador de linguagem C, que são processos do GNU. Ou seja, o Linux é basicamente a união de todos os sistemas anteriores a ele, utilizando o melhor possível em cada sistema operacional. Mas somente em 1992 o Linux é lisenciado sob a lisença GNU e em 1993 teve finalmente sua primeira versão definitiva, o LINUX 1.0, sendo motivado pelas limitações o Minix.

      Logo, o Linux viria a ser o KERNEL que utilizava das aplicações do GNU, por isso seu nome foi modificado para GNU/Linux e desenvolvido por Linus Torvalds.

  • Kernel Linux

    • Basicamente, o kernel Linux é o núcleo do sistema operacional, o qual é responsável por gerenciar e acessar os recursos de hardware, permitindo o acesso dos aplicativos aos recursos do computador.

    Ele pode ser desenvolvido por uma comunidade global (empresas e usuários), como a IBM, Goole, Intel, Samsung e etc. O kernel Linux possui inúmeras versões até chegar na atual e essas diferentes versões estão presentes no site Kernel.org
